Модель предназначенная для определение кто изображён на фотографии (2 категории) и bounding box лиц.

Данные находятся в папке Images, должны разделены на train/valid в которых находятся фотографии и txt файлы (class/xmin/ymin/xmax/ymax)
Обученная модель (3 эпохи) находится в папке Outputs.

Для начала взята VGG16 модель. Работа программы осуществляется без использования tensorflow object detection api.
 
train.py - предназначен для обучения модели
predict.py - предназначен для работы с обученной моделью

в папке lib_image_search:
config.py - предназначен для постановки глобальных переменных (путей, констант и тд)
get_data.py - предназначен для преобразования входных данных
loss_and_metrics.py - собраны кастомные функции метрик и потерь
